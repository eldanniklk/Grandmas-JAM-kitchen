# Grandma’s JAM Kitchen

**JAM is complex. Grandma makes it simple.**

Grandma’s JAM Kitchen is a playful interactive explainer for understanding the basic flow of a JAM service through a visual story with Grandma, Refine Bot and the Sacred Jar.

The demo focuses on the order of the JAM flow:

- Grandma’s Kitchen = JAM Service
- Pre-packed Ingredient Box = Work Package
- Fruit inside the box = Work Item
- Refine Bot = Refine, running the service recipe over each item
- Approved Fruit = Work Result
- Clean Tray = Work Report
- Inspectors = Guarantees / Assurances / Judgments intuition
- Grandma = Accumulate, folding accepted results into state
- Sacred Jar = State
- Delivery Duck = onTransfer

Main idea:

> The box comes first. Refine returns results. Accumulate updates the jar.

This is an educational simplification based on public JAM/Polkadot concepts. Exact JAM implementation details may evolve.
