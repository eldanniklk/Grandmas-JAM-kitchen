# Grandma’s JAM Kitchen

**JAM is complex. Grandma makes it simple.**

Grandma’s JAM Kitchen is a playful interactive explainer for understanding the basic mental model of JAM services through a visual story with Grandma, Refine Bot and the Sacred Jar.

The goal is to make complex Polkadot/JAM concepts easier to understand by turning them into a simple interactive experience:

- Grandma’s Kitchen = JAM Service
- Ingredient Box = Work Package
- Fruit = Work Item
- Refine Bot = Refine
- Clean Tray = Work Result
- Grandma = Accumulate
- Sacred Jar = State
- Delivery Duck = onTransfer

Main idea:

> Refine checks the fruit. Accumulate fills the jar. The jar remembers.

This is an educational simplification based on public JAM/Polkadot concepts. Exact JAM implementation details may evolve.
