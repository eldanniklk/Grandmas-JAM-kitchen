Nerd mode

JAM is the Join-Accumulate Machine. A work package is prepared before Refine and contains one or more work items. Refine runs mostly stateless service code over each item and yields work results. A package's results form a work report, which is checked through guarantees, assurances and possible judgments. Accumulate then folds accepted results into the service state. onTransfer handles incoming messages or tokens from other services asynchronously.

This is an educational simplification. Mental model: package → refine → report → checks → accumulate. Exact JAM implementation details may evolve.
