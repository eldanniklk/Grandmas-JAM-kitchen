Nerd mode

JAM is the Join-Accumulate Machine. A work package is prepared before Refine and contains one or more work items. Refine runs mostly stateless service code over each item and yields work results. A package's results form a work report. Before Accumulate, the report goes through the security layer: guarantees commit validators to the report's correctness, assurances signal that the required chunks are available, and judgments handle reports that may be wrong. Accumulate then folds accepted results into the service state. onTransfer handles incoming messages or tokens from other services asynchronously.

This is an educational simplification. Mental model: package → refine → report → guarantees / assurances / judgments → accumulate. Exact JAM implementation details may evolve.
